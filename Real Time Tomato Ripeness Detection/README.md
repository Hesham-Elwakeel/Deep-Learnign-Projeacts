# 🍅 Real-time Tomato Ripeness Detection

Detect tomato maturity using Deep Learning (MobileNetV2)

## 🎯 Features
- Binary Classification: Immature vs Mature
- Transfer Learning with MobileNetV2
- 95%+ Accuracy
- Real-time prediction
